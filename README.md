# Tic Tac Toe Web Application

A modern and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

- Two Player Mode
- AI Opponent Mode
- Neon UI Design
- Scoreboard
- Winner Popup
- Winning Cell Animation
- Confetti Celebration
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
